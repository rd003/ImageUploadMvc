# ImageUploadMvc
In this project you will learn to perform image related operations (add,update,delete,display). It is a full crud app with actual database. 
Images will be saved locally.

## Tech Stack
- Dotnet 8 mvc
- Dapper
- MSSQL
- Bootstrap 5
